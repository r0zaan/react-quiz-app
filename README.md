A responsive and interactive **React Quiz App** built with **React.js** and **Tailwind CSS**.

Users can:

- Select quiz type: **Generic, Coding, IQ**
- Answer multiple-choice questions
- Track **score** and **high score** (saved in sessionStorage)
- Toggle **Dark and Light mode**
- Play multiple questions (expandable up to 100+)

## Features

- **Question Types:** Generic, Coding, IQ
- **High Score Tracking:** Automatically saves the best score in sessionStorage
- **Dark/Light Mode:** Toggle theme using sun/moon icons
- **Responsive Design:** Works on desktop and mobile devices
- **Expandable Question Bank:** Easily add more questions (up to 100+)

---

## Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **State Management:** React Hooks (`useState`, `useEffect`)
- **Storage:** sessionStorage (for high scores), localStorage (for theme preference)
